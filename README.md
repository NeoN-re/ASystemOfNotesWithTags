# ASystemOfNotesWithTags

# Установка и запуск
Скачайте все файлы проекта в одну папку, откройте файл index.html двойным кликом в любом браузере. Теперь вы можете писать заметки с тегами

# Архитектура
index.html
Главная страница, разметка интерфейса
style.css
Стили и адаптивный дизайн
storage.js
Модуль работы с localStorage
validator.js
Модуль валидации данных
utils.js
Модуль вспомогательных функций
app.js
Основной модуль — логика приложения
